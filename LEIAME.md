# Leia-me

Esta distribuição foi preparada para publicação direta no GitHub Pages sem pastas internas.

## Arquivos essenciais do site

- `index.html`;
- `IdentidadeVisualDoObservatorio.css`;
- `BaseAnaliticaDoVigitel.js`;
- `CatalogoDeIdadeDetalhada.js`;
- `MetodologiaDosIndicadores.js`;
- módulos JavaScript da interface;
- nove arquivos temáticos iniciados por `DadosIdadeDetalhada`.

## Publicação

Extraia o ZIP e envie todos os arquivos para a raiz do repositório. Depois configure o GitHub Pages com a branch `main` e a pasta `/(root)`.

## Reprodução

Os scripts de preparação, recálculo, testes e documentação também estão na raiz. Os microdados oficiais não fazem parte do pacote público; quando necessários, devem ser colocados localmente em uma pasta `Microdados` criada ao lado dos scripts.
