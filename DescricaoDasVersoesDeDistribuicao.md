# Descrição das versões de distribuição

## Versão normal

Distribuição autônoma para consulta, estudo e manutenção. Inclui o painel, os dados publicados, os resultados processados, os scripts, os testes e a documentação. Os microdados individuais não acompanham essa versão para manter o download menor.

## Versão para GitHub

Distribuição preparada para repositório e GitHub Pages. Inclui o painel e os arquivos necessários à publicação, além dos testes e documentos de manutenção. Não inclui microdados individuais nem as tabelas processadas de reprodução, evitando arquivos grandes no repositório.

## Versão para Zenodo

Distribuição de preservação e reprodutibilidade. Inclui tudo o que está nas outras versões, os microdados em CSV, o arquivo XLSX disponível, os dicionários e os resultados processados. Arquivos XLS foram excluídos conforme a organização definida para o acervo.
